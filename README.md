# group3
Что реализовано в приложении(В скобочках указано, кто отвечал за реализацию конкретной задачи)
Приложение реализует алгоритм сортировки "BubbleSort"(Кривов Кирилл)
Сортировка происходит по всем трём полям класса Data(Смирнов Сергей, Кривов Кирилл, Воробьев Владимир)
Класс Data построен с помощью DataBuilder(Смирнов Сергей)
В приложении реализован паттерн стратегия(Воробьёв Владимир) и бинарный поиск(Воробьв Владимир)
Так же выполнена дополнительная сортировка по числовому полю(Кривов Кирилл)
Объекты с четными значениями этого поля сортируются в натуральном порядке, а с нечетными – остаются на исходных позициях.
Написан функционал для записи найденных значений в файл(Воробьёв Владимир)
Реализовано заполнение коллекция посредством стримов(Хузин Марат, Смирнов Сергей)
Коллекция для заполнения кастомная(Хузин Марат)
Реализован Многопоточный метод, подсчитывающий кол-во вхождений элемента N в коллекцию(Пастуханов Петр, Смирнов Сергей)
Так же реализованы тесты к написанному коду(Смирнов Сергей)
Всё приложение смерджено в main ветку(Смирнов Сергей)
